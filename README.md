### Portfolio
# Description
This is the repository of my Portfolio created with Nuxt + tailwind as base with Nuxt/icon and Nuxt/Fonts to give better look.
The goal whit this project was to crete easily maintainable and updatable Portfolio using nuxt and learn how vue works same time. 

## Style decisions
Design followed KISS method (Keep It Simple Stupid), Decisions explained from top to bottom of page.

For Colors i choice to have White and Gray to make the base of website clean and simple. in Night/dark mode i chose for 
dark blue to mimic night + drawings being yelloish white to make look like star/glowing.
Header being just my name plus button to CV is intentional.

For background did not want to have just one color/image or anything animated to keep it simple but stylish.
All White background colors have shadow to highlight area and to keep it not blending into background.
Icons use colors of the company + darker variant of gray on so hover effect is visible. 

For Highlight Projects the Blue is In progress projects and Green for projects that are Completed and Orange is for Paused projects. Filter button to filter to projects 
to clarify between personal and group projects. Hover over the project card shows https url to projects repository or organisation of the project to hint user cards are clickable plus shadow
to make it more visual. Only main Tecnologies used in project are shown not to make project cards too full.

Back button on top of the page has blurry effect + scale to make little magnification effect.
For CV page in xl (+1024px) width monitors intentionally made the pages next to each other to make it quickly and easily readable without to much scrolling on 
smaller screens pages go on top of each other and on small screens text is moved from 60% right to underneath section heading in bold.
Profile section in CV links are intentionally underline + blue to make it easier to know it being link.


All .svg assets are custom and self made.
